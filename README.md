# Project Title 
This is a flashcard app for studying. You can add cards with questions and answers to study, shuffle the cards, or delete them.

## Instructions for Build and Use

Steps to build and/or run the software:

1. If you have a simulator run "xcrun simctl boot "iPhone 15"" then "flutter run"
2. If you want to view it in the browser run "flutter run" and choose option 2 chrome


Instructions for using the software:

1. Open the app "Study with me"
2. Create cards, swipe or press the buttons to navigate, tap to flip the card, use the shuffle and delete buttons to shuffle the cards or delete unwanted cards.
